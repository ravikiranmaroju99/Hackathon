Step 12A — Get the Agent Alias ID

The application will need both the Agent ID and Alias ID to call the agent.

Open CloudShell as opsai-admin and run:

Change the agent id MYZQPFHH7P 

cd ~/opsai-assistant-aws

export AWS_REGION=us-east-1
export AWS_DEFAULT_REGION=us-east-1

AGENT_ID="S64EIXPDJJ"
ALIAS_ID="MYZQPFHH7P"

aws bedrock-agent get-agent-alias \
  --agent-id "$AGENT_ID" \
  --agent-alias-id "$ALIAS_ID" \
  --region "$AWS_REGION" \
  --query 'agentAlias.{AliasName:agentAliasName,AliasID:agentAliasId,Status:agentAliasStatus,AssociatedVersion:routingConfiguration[0].agentVersion}' \
  --output table

You should see:

AliasName: opsai-mvp
AliasID: a 10-character ID
Status: PREPARED

The command lists the aliases connected to your agent.

Run only this command

echo "Region: $AWS_REGION"
